# Animation

## animation-fill-mode

| 参数        | 说明                     |
| --------- | ---------------------- |
| none      | 默认值,回到动画没开始时的状态        |
| backwards | 让动画回到第一帧的状态            |
| forwards  | 让动画停留在最后的状态            |
| both      | 轮流应用forwards和backwards |
