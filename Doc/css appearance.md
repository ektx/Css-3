# css appearance

用于控制元素的渲染效果



### 语法

``` css
button {
    /* 例如:取消按钮的默认渲染效果 */
    -webkit-appearance: none;
    -moz-appearance: none;
}
```

通常,我们使用 `none` 用来取消浏览器默认的效果,这样我们可以自由添加自己的效果
