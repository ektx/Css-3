# demo01

使用自定义的标签来模拟锚点，来展示与控制页面中的跳转内容。

## 优点
- 可以更灵活的控制展示的内容。

## 缺点
- 点击时，页面会进行滚动。
- 跳转会改变页面的 URL，也会记录到历史记录中。
